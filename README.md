## Installation

```console
$ gh extension install koki-develop/gh-grass
```

## Usage

```console
$ gh grass
```

![demo](./docs/demo.gif)

### Specify a User

You can specify a user with the `-u` or `--user` flag.

```console
$ gh grass --user <USERNAME>
```
